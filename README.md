# The Dumpling Hut Web

Modern trilingual website for The Dumpling Hut / La Maison Dumplings in Montreal.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Static localized routes for English, French, and Simplified Chinese

## Routes

- `/en`, `/fr`, `/zh`
- `/en/menu`, `/fr/menu`, `/zh/menu`
- `/en/about`, `/fr/about`, `/zh/about`
- `/en/location`, `/fr/location`, `/zh/location`
- `/en/gallery`, `/fr/gallery`, `/zh/gallery`

## Development

```bash
npm install
npm run dev
npm run lint
npm run build
```
